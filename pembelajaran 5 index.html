Pembelajaran mengubah font

  1. pembelajaran cara mengubah font ini sangat penting. rumusnya adalah (.p1 { font-familiy}) adapun maksud dari p1 disitu adalah bukan termasuk bagian rumus, tapi itu elemen/ tag apa yang ingin kita ubah fontnya. kebetulan di sini saya menggunakan tag p dengan tag class. nah nama dari tag class saya itu adalah p1. oleh karena itu p1 disini bukan termasuk rumus.

  2. adapun cara besar kecilnya huruf adalah menggunakan rumus (.p2 {font-size})

  3. adapun cara bagaimana menebalkan huruf sebenarnya sudah di ajarkan di html, taoi untuk di css, ia memiliki rumus sendiri. yaitu (.p3 {font weight:bold})

  4. adapun mengubah font menjadi italicsebenarnya juga sudah di ajarkan di html. tapi di css sndiri dia memiliki rumus, yaitu (.p4 {font-style:italic}).
