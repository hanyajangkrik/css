css merupakan singkatan dari Cascading Style sheet

bagaimana cara menghubungkan html dengan css?

  1. inline style css.

    2. adapun rumusnya adalah: h1 style="color:blue(warnanya bebas);"> judul artikel /h1

    contoh:

    judul artikel

    mengapa warnyanya berubah menjadi warna biru. karena kita menggunakan rumu inline style css

  2. internal css

    3. adapun rumusnya adalah:

    < style >
    h1 { color : blue; }

    < style >

    Perlu diketahui untuk rumus yang internal css ia hanya berlaku di tag head, bukan di body.

    ada di atas ( liatnya lewat vsc)
