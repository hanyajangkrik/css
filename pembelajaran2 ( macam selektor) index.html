SELEKTOR

selektor adalah sebuah pola yang digunakan untuk mencari sesuatu tab / elemen yang akan diberikan style.

adapun macam macam dari selektor dibagi menjadi 3 macam selektor:

  1. tag selektor

    2. adapun fungsi dari tag selektor ini adalah ia berguna untuk men tag elemen elemen atau tab yanng di gunakan

    contoh:

    ketika di file html kita terdapat elemen p atau h1 dan macam mcam elemen lainnya, ketika kita ingin merubah warna, maka cukup meggunakan tag selektor ini, di mana kita kembali membuat file baru ( file css) kemudian di file itu kita menandai elemen elemen yang ingin kita tandai dengan type.

    aku suka makan garam

  2. class selektor.

    3. adapun fungsi dari class selektor ini adalah untuk membedakan warna teks meskipun yang di tag itu satu elemen yang sama.

    caranya:

    cukup hanya letakan elemen yang mau kita buat, nah apabila misalnya nih elemenn trs adalah elemen p, dan tentu elemen p ini g hanya satu yang kita gunakan...bisa jadi malah banyak pengulangan elemen p ini. dan ternyata kita ingin dari sekian banyak elemen p tersebut warnya artikel di elemen p tadi warnanya berbeda-beda. maka, caranya adalah cukup letakan style CLASS di samping elemen p ataupun elemen yanng ingin kita buat. dan setelah itu kasih nama di tanda (""). setelah iotu pergi ke file css yang kita buat untuk mebuat warna. setelah itu untuk memulai selektor class ini, maka awali dengan titik (.) terlebih dahulu, lalu setelahnya masukan nama yang kita letakan di samping elemen elemen tadi.

    aku suka makan garam

    aku sangat suka makan garam

  3. id selektor

    4. adapun id selektor ini, ia sebenarnya penggunaannya sama saja dengan class selektor. hanya saja kalau class selektor ia di awali dengan titik (.), untuk id selektor ini ia diawali dengan pagar (#). caranya ia juga sama dengan class selektor yang membedakan hanya penulisann awalnya saja. antara titik (.) dengan pagar (#).
